About Me

I am a Computer Science and Mathematics student at Virginia Tech. Currently, I'm freelancing as a web developer. I spend my spare time programming several projects, playing PC games, or enjoying a good book or movie. My music tastes vary, but mostly listen to symphonic metal.

I am well versed in GNU/Linux and almost as knowledgeable in the Windows suite. I know Java, C, python, HTML, CSS, JavaScript, and bash and I am in the process of learning Clojure (which this site was written in), Ruby, C++, and x86 assembly. I haven't decided on my favourite language yet, but I will say the code I've been happiest with was written in C.

The Minecraft server I run is on downtime while the players wait for me to finish writing a plugin, ResearchCraft, for the server, but it's been mostly up for the past year and a half. I'm also working with a few students at Tech on various programming projects, none of which are substantial, but are mostly to expand our knowledge and improve our familiarity with different programming aspects.

About This Site

This website was was written in Clojure on Ring and Compojure. It also uses Postal for the email form. The HTML portion uses JQuery and sweet-justice. It's hosted on the heroku cloud. The source is available on github

I put this together to experiment with functional languages. Also because it was long due that I made my own website. Some of the code was patched together from various sources, but all the HTML, CSS, and any JavaScript that wasn't already mentioned was written by me from scratch. Also I don't own rights on the logos I used. The fonts are Nova Slim and Dancing Script.
